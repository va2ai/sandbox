# GenStudio AI

Mobile-first PWA for generative AI design using xAI Grok and Google Gemini APIs.

## Repository
- **Repo**: https://github.com/va2ai/sandbox
- **Part of**: sandbox megarepo (first project)

## Tech Stack
- Next.js 16 (App Router, Turbopack)
- Tailwind CSS 4 + shadcn/ui
- Zustand (state management with localStorage persistence)
- TanStack Query v5
- Framer Motion (animations)
- next-pwa (PWA support)

## Key Directories
- `/src/app` - Next.js pages and API routes
- `/src/components/generation` - Generation UI components
- `/src/lib/api` - xAI and Gemini API clients
- `/src/lib/stores` - Zustand stores
- `/public/icons` - PWA icons (SVG)

## API Integration
- **xAI Grok**: Text-to-image, image editing, video generation
- **Gemini**: Gemini 2.0 Flash (native image gen), Imagen 3.0

## Available Modes
- T2I: Text to Image
- I2I: Image to Image (editing)
- T2V: Text to Video
- I2V: Image to Video

## Commands
```bash
npm run dev     # Development server
npm run build   # Production build
npm run start   # Start production server
```

## PWA Setup
- Manifest: `/public/manifest.json`
- Icons: `/public/icons/icon-*.svg`
- Service worker generated by next-pwa in production

## API Keys
Users configure API keys in Settings page:
- xAI: https://x.ai/api
- Gemini: https://aistudio.google.com/apikey

Keys are stored in localStorage (browser-side only).

## Known Issues
- next-pwa types are outdated for Next.js 16, using custom type definitions
- PWA disabled in development mode
